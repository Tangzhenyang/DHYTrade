选择免费股票行情API

不同API覆盖的市场和特性不同

A

新浪财经 API

覆盖沪深A股、港股，免费无限制，实时性好。适合纯A股场景。

B

东方财富 API

覆盖沪深A股、港股、美股，数据全面，免费。需要处理请求频率限制。

C

Yahoo Finance API

全球市场覆盖，免费但有速率限制。对A股实时性不如国内源。

D

组合策略（推荐）

默认用新浪/东方财富覆盖A股，同时预留Yahoo Finance接口用于美股扩展。后端通过统一接口抽象，方便切换。

推荐理由

方案D（组合策略）

  • A股走国内源，实时性最好
  • 后端做接口抽象，随时换数据源
  • 为未来扩展美股留好通道

需要考虑

  • 需要实现一个简单的行情适配层
  • 不同数据源格式不同，需要做标准化

用户权限模型

谁可以使用系统？怎么注册？

A

双角色：管理员 + 普通用户

管理员：管理仓位、发布公告。普通用户：查看仓位、使用跟仓计算器。注册默认普通用户。

B

三角色：超级管理员 + 操作员 + 用户

超级管理员管用户，操作员负责日常买卖记录，用户跟仓。适合多人协作团队。

注册方式

C

开放注册

任何人都可以注册，注册后为普通用户。管理员账号由系统初始化创建。

D

仅邀请注册

管理员生成邀请链接或邀请码，只有被邀请的人才能注册。更私密可控。
